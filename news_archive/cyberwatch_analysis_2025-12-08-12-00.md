### Summary
- [Android Malware FvncBot, SeedSnatcher, and ClayRat Gain Stronger Data Theft Features](https://thehackernews.com/2025/12/android-malware-fvncbot-seedsnatcher.html): Researchers have identified new Android malware families, FvncBot and SeedSnatcher, along with an upgraded version of ClayRat. - [Sneeit WordPress RCE Exploited in the Wild While ICTBroadcast Bug Fuels Frost Botnet Attacks](https://thehackernews.com/2025/12/sneeit-wordpress-rce-exploited-in-wild.html): A critical remote code execution vulnerability (CVE-2025-6389) in the Sneeit Framework plugin for WordPress is being actively exploited. The vulnerability has a CVSS score of 9.8 and affects all versions prior to 8.4.
- [MuddyWater Deploys UDPGangster Backdoor in Targeted Turkey-Israel-Azerbaijan Campaign](https://thehackernews.com/2025/12/muddywater-deploys-udpgangster-backdoor.html): The Iranian hacking group MuddyWater has been using a new backdoor called UDPGangster for cyber espionage against targets in Turkey, Israel, and Azerbaijan, allowing remote control of compromised systems.
- [Critical Apache Tika Vulnerability Leads to XXE Injection](https://www.securityweek.com/critical-apache-tika-vulnerability-leads-to-xxe-injection/): A critical vulnerability in Apache Tika allows attackers to perform XML External Entity (XXE) injection attacks via crafted XFA files in PDF documents.
- [Exploitation of React2Shell Surges](https://www.securityweek.com/exploitation-of-react2shell-surges/): There has been a notable increase in attempts by threat actors to exploit the React vulnerability CVE-2025-55182.
